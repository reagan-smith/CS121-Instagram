# README

### Description
This website was constructed as an intro to learning Ruby on Rails. It was desgined and modeled after this tutorial (https://www.youtube.com/watch?v=MpFO4Zr0EPE). It is a basic instagram like site, with the ability to post photos with captions and has a main page where these photos are displayed, as well as user accounts (login, register, logout).
